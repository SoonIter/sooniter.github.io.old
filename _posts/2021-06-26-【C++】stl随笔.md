###二分搜索
lower_bound(vc.begin(),vc.end(),num)：从数组的begin位置到end-1位置二分查找第一个大于或等于num的数字，找到返回该数字的地址，不存在则返回end。通过返回的地址减去起始地址begin,得到找到数字在数组中的下标。
```cpp
vector<int>vc;//必须确定为有序

...
int find(num){
	return lower_bound( vc.begin(),vc.end(),num)-vc.begin();
}


```
